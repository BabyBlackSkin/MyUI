Message 消息提示

常用于主动操作后的反馈提示，用于展示操作结果、系统消息或警告信息。消息在指定时间后自动消失，也可手动关闭。

基础用法

从顶部出现，3 秒后自动消失。

Info 消息 Primary 消息 Success 消息 Warning 消息 Error 消息

不同状态

提供 primary、success、warning、info、error 五种类型，通过 type 字段区分。

Info Primary Success Warning Error

可关闭消息提示

默认情况下，消息在 3 秒后自动关闭且不显示关闭按钮。
设置 showClose: true 可显示关闭按钮；设置 duration: 0 可禁用自动关闭。

可关闭（不自动消失） 无关闭按钮（3秒消失）

自定义时长

通过 duration 参数控制消息显示时长（毫秒）。设置为 0 或负数则不自动关闭。

显示 10 秒 永不消失

消息位置（Placement）

通过 placement 参数控制消息出现的位置，支持： top、top-left、top-right、 bottom、bottom-left、bottom-right，默认为 top。

顶部居中（默认） 顶部左侧 顶部右侧 底部居中 底部左侧 底部右侧
