MessageBox 消息弹出层

模拟实现 ElementPlus 的 MessageBox 组件，支持消息提示和确认消息功能。

基础用法

调用 messageBox.alert 方法可以打开消息提示弹窗。

打开 Alert

消息类型

支持 info、success、warning、error 四种消息类型。

Info Success Warning Error

确认消息

调用 messageBox.confirm 方法可以打开确认消息弹窗。

打开 Confirm

自定义按钮

可以自定义按钮的文本和类型。

自定义按钮

回调函数

可以通过回调函数处理用户的操作结果。

带回调的确认

代码示例

// 基础用法
messageBox.alert('这是一段内容', '标题');

// 消息类型
messageBox.info('这是一条信息提示', '消息');
messageBox.success('恭喜你，这是一条成功消息', '成功');
messageBox.warning('这是一条警告消息', '警告');
messageBox.error('这是一条错误消息', '错误');

// 确认消息
messageBox.confirm('此操作将永久删除该文件, 是否继续?', '提示', {
    confirmButtonText: '确定',
    cancelButtonText: '取消',
    type: 'warning'
}).then(() => {
    console.log('用户点击了确定');
}).catch(() => {
    console.log('用户点击了取消');
});

// 带回调的确认
messageBox.confirm('此操作将永久删除该文件, 是否继续?', '提示', {
    callback: function(action) {
        if (action === 'confirm') {
            console.log('用户点击了确定');
        } else {
            console.log('用户点击了取消');
        }
    }
});
